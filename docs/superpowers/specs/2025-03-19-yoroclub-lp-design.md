# 養老昆虫クラブ ランディングページ 設計書

## 概要

養老昆虫クラブの公式ランディングページ。佐藤卓デザインのピクセルアート風ロゴの世界観を活かし、WebGLアニメーションでインタラクティブな体験を提供する1ページサイト。

## ゴール

- クラブの存在と活動内容を伝える
- 開催中の「養老孟司と小檜山賢二の虫展」を告知する
- テーマソング「むしのいどころ」を紹介する
- SNSへの導線を設置する（アカウント未作成のためプレースホルダー）

## デザイン方針

**白背景 + 黒アクセント**。ロゴの白黒ピクセルアートの世界観を全体に展開する。

## WebGL ピクセルアニメーション

Three.jsを使用してロゴのピクセルを粒子として制御する。

### 体験フロー

1. **ページ読み込み時**: 黒いピクセル粒子が画面全体にランダムに散布
2. **集合アニメーション（1.5〜2秒）**: 粒子が滑らかに移動し、ロゴの形を形成。イージング: ease-out
3. **テキスト表示**: 「養老昆虫クラブ」がHTMLオーバーレイとしてフェードイン（WebGLキャンバスの上にCSS positionで重ねる）
4. **背景浮遊**: ロゴ形成に使われなかった余剰粒子が背景でゆっくり浮遊（速度: 0.2〜0.5px/frame）
5. **インタラクション**: マウスホバー/タッチでピクセルが反発（半径100px、バネモデルで元の位置に戻る）

### パフォーマンス対応

| デバイス | 粒子数 |
|---|---|
| デスクトップ | 最大2000個 |
| モバイル | 最大500個 |

- `navigator.hardwareConcurrency` と画面サイズで自動判定
- タッチイベント対応（touchstart, touchmove）
- `prefers-reduced-motion` 有効時: アニメーションなし、静的ロゴ表示
- WebGL非対応時（`canvas.getContext('webgl')` が null）: 静的ロゴ画像にフォールバック

## ロゴ画像の扱い

- 提供済みの佐藤卓デザインのピクセルアート風ロゴ画像を `public/logo.png` に配置
- WebGLアニメーション用: Canvas APIで画像を読み込み、黒ピクセル（閾値: RGB各128以下）の座標を抽出してパーティクルの目標位置とする
- フォールバック用に静的PNG画像も `<img>` で配置（WebGL非対応時に表示）
